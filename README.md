# Game Project

To play the game you must follow the instructions:

```sh
cd game
python3 main.py 
```

# App Project

To use the app follow the instructions:

```sh
cd app
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
